### Managing AI Agent Teams

A Udemy course repository covering practical techniques for building and managing AI agent workflows using Claude Code.

---

## What's in This Repository

### Course Projects

**`COURSE_PROJECTS/MARKETING_DATASET/`**
A synthetic 12-month (2024) UK marketing dataset used for hands-on exercises in data analysis, dashboarding, and AI-assisted reporting. Includes five CSV files:

- **Site Traffic** — Daily sessions, users, and conversions across 7 channels (Organic, Paid, Social, Direct, Email, Referral, Display)
- **SEO Keywords** — Monthly keyword rankings, impressions, click-through rates, and search intent classification
- **Paid Campaigns** — Performance metrics for Google Ads, Meta, and LinkedIn campaigns (spend, CPC, ROAS, conversions)
- **Page Performance** — Page-level analytics including Core Web Vitals (LCP, CLS, FID)
- **Audience Demographics** — ~22,000 rows of segmentation data by country, device, age group, and gender

### Claude Code Skills

**`.claude/skills/branding/`**
A reusable Claude Code skill that demonstrates how to set up persistent brand instructions and use them to generate branded PowerPoint presentations and WordPress CSS. Covers:
- Structuring `CLAUDE.md` and `BRAND.md` for brand consistency
- Generating `.pptx` files with PptxGenJS
- Producing CSS custom properties from brand tokens
